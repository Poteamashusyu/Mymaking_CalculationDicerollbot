詳細説明

・タイムリーボーナス*

・タイムリーボーナスは数時間間隔で配られるガチャコインイベントです。
・時間経過後にガチャを回すとタイムリーボーナスが適用され、専用の抽選によって配当を決定します。
・タイムリーボーナスはガチャを1回以上回していてコインデータがあるプレイヤー全員に配られます。
その際、タイムリーボーナスを行ったプレイヤーは配当が2倍になります。
・ガチャを一定時間(48時間程度)回さないと受け取れるボーナス量が20%に減少します。
<イメージ>

・リザーブ(予備)コイン**

・リザーブコインはボットオーナー(運営者)がガチャラインアップ変更時等に設定します。
リザーブコインが設定されると各々のプレイヤーは次の条件にしたがってコイン・リザーブコイン枚数が変動します。

※設定枚数500枚のとき

設定枚数より多いプレイヤー>設定枚数を超えるコインがリザーブコインに移ります。
例:
コイン: 600　 >>> 　コイン: 500
リザーブコイン: 0　 >>> 　リザーブコイン: 100
設定枚数以下のプレイヤー>特に変化はありません。
例:
コイン: 450　 >>> 　コイン: 450
リザーブコイン: 0　 >>> 　リザーブコイン: 0

・リザーブコインは普通のコインのように使用は出来ません。
ただし次の条件に従ってタイムリーボーナスが配られた際に一定の枚数(100枚前後)のリザーブコインがコインなります。

所持リザーブコイン枚数が
設定枚数以上のプレイヤー		>設定枚数分のリザーブコインがコインになります。
所持リザーブコイン枚数が
1枚以上、設定枚数より少ないプレイヤー		>持っているリザーブコイン全てがコインになります。
リザーブコインがないプレイヤー>特に変化はありません。

投票ギャンブル
「_cd gamble ...」,「_cd gam ...」
機能ガチャコインを使用し、投票制ギャンブルを行うことが出来る。
記述 _cd gamble ... (「_cd gam ...」)

bet <パスコード*> <ベット番号> <ベット額>>受け付け中(開始前)のギャンブルセッション(以下、セッション)にベットします。(*: 後述)
detail(det)>準備中・プレイ中のセッションを検索し詳細を表示します。
※実行すると新たにコマンド案内が出るのでそれに従ってください。
end <勝利選択肢(名前)>>自身が主催するプレイ中セッションの結果を確定させてそのセッションを終了します。
help>投票ギャンブルのコマンド詳細を表示します。
open <タイトル> <内容1,2,...,6>>自身が主催する新しいセッションを作成します。
start>準備中の自身のセッションを開始(プレイ状態に)します。開始以降、新たにベットすることが出来なくなります。

例 ⓵「_cd gamble bet 4649 1 100」 => (パスコード:4649の準備中セッションの1番に100コインベットする。)
⓶「_cd gamble open 誰が勝つ？ Aさん Bさん」 => (タイトル:誰が勝つ？で選択肢: Aさん, Bさんのセッションを作成します。)
⓷「_cd gamble start」 => (自身の準備中セッションを開始(ベット締め切り)します。)
⓸「_cd gamble end Aさん」 => (自身のプレイ中セッションの当選を「Aさん」にしてセッションを終了します。)

・ギャンブルパスコードについて*

・このパスコードは0~9999で生成され、セッションを識別して選択するために使われます。

◆ギャンブルの仕組みについて

・この投票ギャンブルは競馬や競艇などに近い(実際とは異なります)システムを採用しています。
実際の配当確定までの過程はこちら...

・配当確定までの過程 (※当選したのが「No.2」の場合)

1. ベット合計を当選した番号にベットした人に分配します。
2. 一人一人の分配する比率はそれぞれのプレイヤーが当選番号にベットした量に比例します。
<公式1> (その番号が当選したときの獲得倍率) = (ベット合計額) / (その番号の総ベット額)
掛け金合計各番での掛け金合計-ベット内訳-
1000枚
No.1: 400枚 (2.5倍)(今回は関係ないので省略)...
No.2: 100枚 (10倍)Aさん: 35枚, Bさん: 35枚, Cさん: 30枚
No.3: 500枚 (2.0倍)(今回は関係ないので省略)...

  
  35%35%30%
<公式2> (自身の配当(当選額)) = (ベット合計額) * {(自身が当選した番号にベットした額) / (当選した番号の総ベット額)}
当選者計算式
Aさん 350枚 = 1000 * (35 /100)
Bさん 350枚 = 1000 * (35 /100)
Cさん 400枚 = 1000 * (40 /100)
